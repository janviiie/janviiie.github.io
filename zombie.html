About Zombieland

Zombieland – A Thrilling Zombie Shooter Game Zombieland is an action-packed zombie survival game developed in Unity Engine using C#. It features an intense first-person shooter (FPS) experience where players must fight off waves of zombies using multiple weapons while navigating through different maps.

Game Features:

Main Menu with Four Options:

Play – Choose a map and start the action-packed gameplay.
Settings – Adjust mouse sensitivity and game difficulty for a personalized experience.
Maps – Currently includes two playable maps, with a third map in development.
Exit – Quit the game anytime.

  6.  
  7. 	float tInnerPosToCircleLength = tInnerPosToCircle.Size();
  8.  
  9. 	/* Normalized direction from player to circle */
  10. 	FVector tInnerPosToCircleDirection = tInnerPosToCircle;
  11. 	tInnerPosToCircleDirection.Normalize();
  12.  
  13. 	if (tInnerPosToCircleLength > mProps->Radius + THRESH_POINTS_ARE_NEAR)
  14. 	{
  15. 		/* Player has moved outside of the circle, calculate the difference and add it to the position */
  16. 		float tMoveDeltaLength = tInnerPosToCircleLength - mProps->Radius;
  17. 		FVector tMoveDelta = tInnerPosToCircleDirection * tMoveDeltaLength;
  18. 			
  19. 		/* Calculate the delta rotation */
  20. 		float tPushAngle = FMath::RadiansToDegrees(FMath::Acos(FVector::DotProduct(tInnerPosToCircleDirection, mRotation.GetRightVector())));
  21. 		tPushAngle = -(tPushAngle - 90);
  22.  
  23. 		/* Add position and rotation */
  24. 		mPosition += tMoveDelta;
  25. 		mRotation *= FQuat::MakeFromEuler(FVector::UpVector * (tPushAngle / mProps->RotationTightness));
  26. 	}
  27.  
  28. 	mInnerPosition = aNewPos;
  29. }

Weapons & Gameplay:

Players can switch between four different guns to eliminate zombies.
As time progresses, zombies spawn in greater numbers, increasing the challenge.
If zombies get too close, they attack the player, making survival even more intense.

Advanced AI & Movement System

The zombies are designed with AI-driven movement, ensuring unpredictable attacks and engaging combat.
Smooth and responsive character controls for a seamless FPS experience.
With immersive gameplay, strategic shooting mechanics, and ever-increasing challenges, Zombieland delivers an exhilarating survival experience.
